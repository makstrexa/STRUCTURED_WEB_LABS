У ході виконання даної лабораторної роботі ми нагадали основні інструменти для використання бібліотеки jQuery UI та JavaScript у HTML-документі. Ми опрацювали теорію про використання в сценаріях JavaScript бібліотек крос-браузерних інтерфейсів методів dom та бібліотеки jQuery UI. Використовуючи вивчений матеріал на практиці при виконанні завдань, ми також придбали практичні навички роботи з jQuery UI, додавання до сайту динамічні елементи цієї бібіліотеки, плагін галереї UniteGallery, кнопки соціальних мереж, інформери та інше. Запустили даний документ у браузері для перевірки та переконалися, що все працює коректно.
