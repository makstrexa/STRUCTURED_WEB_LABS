Лабораторна робота №1

Користувач, перебуваючи на сайті, може побачити світлину самого університета, форму для реєстрації нового користувача та входу, а також основне меню.

Натисканням "Додому" він у подальшому повертатиметься на головну сторінку сайту.

Кнопка "Контактів" показуватиме доступні варіанти зв'язку, а саме: номери мобільних телефонів, теги у Телеграмі.

Переходячи у закладинку "Про нас" користувач зможе отримати офіційну адресу нашого університета, а також основну інформацію стосовно організаційних моментів.

Кожний бажаючий зможе переглянути списки студентів та загальний розклад заняття груп, що надасть змогу оцінити планування навчального процесу.

Сайт має відповідати таким функціональним вимогам:

- гість повинен мати можливість переглядати сторінки сайту;
- гість повинен мати можливість здійснювати пошук у розділі "Меню" ;
- адміністратор повинен мати такі самі можливості, як і у користувача ;
- адміністратор повинен мати змогу оновлювати інформацію на сайті.

Сайт має відповідати таким нефункціональним вимогам:

- для оформлення дизайну сайту використовувались мови HTML та CSS;
- код для оформлення дизайну сайту був написаний у текстовому редакторі Sublime Text;
- інтерфейс повинен бути легким та зручним у використанні;
- сайт повинен бути кросплатформеним.
